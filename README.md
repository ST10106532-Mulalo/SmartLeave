## 📋 Overview
SmartLeave is a comprehensive leave management system for modern enterprises. It streamlines leave requests, late arrivals, and overtime management with role-based access and real-time notifications.

## ✨ Features

### 👥 Employees
- Apply for leave with document upload
- Report late arrivals with photo evidence
- View overtime opportunities with red flash notification
- Track leave history and status

### 👔 Managers
- Approve/reject leave requests
- Review late arrival reports
- Post overtime opportunities
- View team member details

### 🔧 Admins
- User approval workflow
- Employee directory management
- System-wide reports
- Role management

## 🏗️ Tech Stack
- **Frontend:** Angular 21 with Cyberpunk UI
- **Backend:** .NET 10 Web API
- **Database:** SQL Server
- **Authentication:** JWT with Role-based access

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- .NET 10 SDK
- SQL Server 2022

### Installation
1. Clone the repository
2. Run `database/SmartLeaveDB.sql` in SQL Server
3. Start backend: `cd SmartLeaveAPI && dotnet run`
4. Start frontend: `cd frontend && ng serve`
5. Login: `admin@company.com` / `12345678910`

### One-Click Launcher
Double-click `start.bat` to run both backend and frontend simultaneously!
